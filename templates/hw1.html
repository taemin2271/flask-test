AI 웹캠을 활용한 실시간 자세 교정 및 건강 데이터 분석 서비스

빅데이터 수업 프로젝트 계획서 (통계학과 홍태민)

1) Problem Statement Canvas

고객

컴퓨터 앞에서 장시간 작업하는 20-40대 학생, 직장인, 개발자

컨텍스트

하루 평균 8시간 이상 의자에 앉아 모니터를 보며 업무나 학습에 집중하는 환경

근본 문제

  • 자신의 자세가 나빠지는 순간을 인지하기 어려움
  • 교정의 필요성은 알지만 지속적인 실천이 힘듦

감정적 영향

  • 만성 피로
  • 집중력 저하
  • 통증으로 인한 스트레스
  • 무기력감

정량적 영향

  • 목/허리 통증으로 인한 병원비 지출
  • 업무 생산성 저하
  • 삶의 질 하락

대안

  • 고가의 인체공학 의자 구매
  • 주기적인 스트레칭 알림 앱
  • 물리치료/도수치료

대안의 단점

  • 높은 비용 부담
  • 단순 알림은 쉽게 무시됨
  • 문제 발생 후의 사후 처리
  • 근본적인 습관 교정 불가

2) 솔루션 (3줄)

  • Line 1. 별도 장비 없이 컴퓨터 웹캠만으로 사용자의 거북목, 어깨 비대칭 등 잘못된 자세를 실시간으로 감지합니다.
  • Line 2. 자세가 무너지면 즉각적인 알림을 주고, 하루 동안의 자세 데이터를 분석하여 시각화된 대시보드를 제공합니다.
  • Line 3. 데이터 분석 결과를 바탕으로 사용자에게 가장 필요한 스트레칭 영상을 맞춤형으로 추천하여 건강한 습관을 만들어줍니다.

3) 기능 명세

  • 사용자 초기 자세 측정(캘리브레이션) 기능
  • 실시간 자세 모니터링 및 '자세 점수' 계산
  • 자세 교정 알림 (설정된 기준 초과 시)
  • 개인화된 데이터 대시보드 (시간대별/주간별 통계)
  • 맞춤형 스트레칭 영상 추천
  • 사용자 회원가입 및 데이터 관리

4) 간단한 UX 흐름

  • 회원가입/로그인 → (최초 1회) 바른 자세 측정 및 등록
  • 웹캠 접근 권한 허용 → 백그라운드에서 실시간 자세 분석 시작
  • 자세 나빠지면 알림 수신 → 자세 교정
  • '내 대시보드' 접속 → 자세 데이터 리포트 확인 및 추천 스트레칭 수행

5) 와이어프레임 (스케치)
